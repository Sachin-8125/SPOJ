# 🌐 SPOJ Solutions

A collection of **competitive programming solutions** from [SPOJ (Sphere Online Judge)](https://www.spoj.com/), written in **C++17**.

> Problems span Basics, Tutorial, and Classical categories — covering a wide range of algorithmic topics.

---

## 📂 Repository Structure

Solutions are organized into category-based folders, mirroring SPOJ's own problem classification:

```
SPOJ/
├── SPOJ - Basics/          # Beginner-friendly problems
├── SPOJ - Tutorial/        # Guided/educational problems
├── SPOJ - Classical/       # Classic algorithmic challenges
├── Alphacode.cpp
├── Brackets.cpp
├── Factorial.cpp
├── The_Next_Palindrome.cpp
└── ...                     # Additional root-level solutions
```

### Category Guide

| Folder | Description |
|---|---|
| `SPOJ - Basics` | Fundamental problems for building core skills |
| `SPOJ - Tutorial` | Step-by-step learning problems covering standard algorithms |
| `SPOJ - Classical` | Well-known SPOJ problems frequently used for practice and reference |

## 🚀 How to Run

### Prerequisites
- GCC compiler with C++17 support (`g++`)

### Compile & Execute

```bash
# Compile
g++ -std=c++17 -O2 -o solution Factorial.cpp

# Run
./solution
```

### With Input File

```bash
g++ -std=c++17 -O2 -o solution The_Next_Palindrome.cpp
./solution < input.txt
```

## 🔍 Finding a Problem

Each file is named after the problem title. To find the problem statement on SPOJ, search by name:

```
https://www.spoj.com/problems/<PROBLEM_CODE>/
```

**Example:** For `Factorial.cpp` → https://www.spoj.com/problems/FCTRL/

> **Tip:** SPOJ uses short uppercase problem codes (e.g., `FCTRL`, `PALIN`, `NXTPAL`). Search the problem title on SPOJ to find the matching code.
